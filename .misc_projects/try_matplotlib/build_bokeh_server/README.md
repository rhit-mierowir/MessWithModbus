To Run this server, run the following command:

``` bash
poetry run ./.misc_projects/try_matplotlib/build_bokeh_server/Render_Graphs.py
```
Use ctrl+c to exit.

This should automatically open up a website on a local internet browser to see the plot.

To reconfigure ports and other things, loot at the end of the `Render_Graphs.py` file in the `if __name__ == '__main__':` section.
